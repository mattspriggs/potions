# Boilerplate: Knex.js

## Setup

### What's included

Phase 1 boilerplate project with:

 - Express
 - Knex
 - SQLite3
 - Express Handlebars
 - body-parser
 - Jest
 - nodemon

### Installation

```
npm install
npm run knex migrate:latest
npm run knex seed:run
npm run dev
```

Be sure to check out the other npm scripts too.
